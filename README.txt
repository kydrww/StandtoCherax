StandtoCherax - Vehicle Converter
==================================

Developed by @notkydrw
Get support at: http://discord.gg/qVEwtkz2Ag

HOW TO USE:
-----------
1. Run StandtoCherax.exe
2. Click "Open Stand Vehicle" and select a folder containing .txt vehicle files
3. Click "Convert to Cherax .json(s)"
4. Find your converted files in the Cherax_<FolderName> folder

REQUIRED FILES:
---------------
Make sure these files are in the same folder as StandtoCherax.exe:
- base.json (already included)(do not change anything)
- vehiclelist.ini (already included)(do not change anything)

The executable is self-contained - no installation needed!
